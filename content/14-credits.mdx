# Credits

veslx is built on top of amazing open source projects.

## Core

| Library | Description |
|---------|-------------|
| [Vite](https://vite.dev) | Next-generation frontend tooling |
| [React](https://react.dev) | UI library |
| [Bun](https://bun.sh) | JavaScript runtime and package manager |
| [TypeScript](https://www.typescriptlang.org) | Type safety |

## Content

| Library | Description |
|---------|-------------|
| [MDX](https://mdxjs.com) | Markdown with JSX components |
| [remark-gfm](https://github.com/remarkjs/remark-gfm) | GitHub Flavored Markdown |
| [gray-matter](https://github.com/jonschlinkert/gray-matter) | Frontmatter parsing |
| [Shiki](https://shiki.style) | Syntax highlighting |

## Math

| Library | Description |
|---------|-------------|
| [KaTeX](https://katex.org) | Fast math typesetting |
| [remark-math](https://github.com/remarkjs/remark-math) | Math in Markdown |
| [rehype-katex](https://github.com/remarkjs/remark-math/tree/main/packages/rehype-katex) | KaTeX rendering |

## Styling

| Library | Description |
|---------|-------------|
| [Tailwind CSS](https://tailwindcss.com) | Utility-first CSS |
| [shadcn/ui](https://ui.shadcn.com) | Accessible UI components |
| [Radix UI](https://www.radix-ui.com) | Unstyled, accessible primitives |
| [Lucide](https://lucide.dev) | Icon library |

## Routing & State

| Library | Description |
|---------|-------------|
| [React Router](https://reactrouter.com) | Client-side routing |
| [date-fns](https://date-fns.org) | Date utilities |

## Build Tools

| Library | Description |
|---------|-------------|
| [cac](https://github.com/cacjs/cac) | CLI framework |
| [pm2](https://pm2.keymetrics.io) | Process manager for daemon mode |
| [chokidar](https://github.com/paulmillr/chokidar) | File watching |
