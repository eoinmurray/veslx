# Quickstart

Veslx serves MDX files from a folder with almost no setup.

## Prerequisites
- [Bun](https://bun.sh)

## Install
```bash
bun install -g veslx
```

Or run without installing:

```bash
bunx veslx serve
```

## Run
```bash
veslx server <your-docs-folder>
```

or

```bash
cd <your-docs-folder>
veslx serve
```

Open http://localhost:3000.

## Build
```bash
veslx build
```

Static site goes to `dist` by default.

## A minimal structure
```
docs/
├── 00-welcome.mdx
├── 01-quickstart.mdx
├── guide/
│   └── README.mdx
└── images/
    └── diagram.png
```
