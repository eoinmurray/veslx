## VeslxFrontMatter
Renders a styled header from frontmatter.

```mdx
<VeslxFrontMatter />
```

## VeslxPostList
Lists pages in the current folder.

```mdx
<VeslxPostList />
```

Group content by globs (useful on a custom homepage):

```mdx
# Getting Started
<VeslxPostList globs={["00-*", "01-*"]} />

# Tutorials
<VeslxPostList globs={["tutorials/*"]} />
```

Props:

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| globs | string[] | null | Filter by file glob |

## VeslxPostListItem
Build custom lists.

```mdx
<VeslxPostListItem
  title="Getting Started"
  description="Intro"
  linkPath="/docs/getting-started"
  date={new Date("2025-01-15")}
/>
```

Props:

| Prop | Type | Notes |
| --- | --- | --- |
| title | string | required |
| description | string | optional |
| date | Date | optional |
| linkPath | string | required |
| isSlides | boolean | optional |

## VeslxCopyButton
Adds a copy button for arbitrary content.

```mdx
<VeslxCopyButton>
  <code>curl https://example.com</code>
</VeslxCopyButton>
```

Props:

| Prop | Type | Notes |
| --- | --- | --- |
| children | ReactNode | required |
