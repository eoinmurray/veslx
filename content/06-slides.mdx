## File naming
Any file named `*.slides.mdx` or `SLIDES.mdx` renders as a slide deck.

## Separators
Use `---` on its own line to start a new slide:

```mdx
# Slide 1

---

# Slide 2
```

## Navigation
- Arrow keys for slide changes
- Scroll to move between slides

## Configuration
Disable scroll snap:

```yaml
slides:
  scrollSnap: false
```
