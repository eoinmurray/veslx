# Parameter Tables

The `ParameterTable` component can display YAML and JSON config files. It now supports glob patterns to show multiple files at once.

## Single File

Display a specific config file:

<ParameterTable path="./experiment-001.yaml" />

## Multiple Files with Glob

Use a glob pattern to display all matching files. Each file is shown with its filename as a label:

<ParameterTable path="./*.yaml" />

## Filtered Keys

You can also filter which keys to show across all matching files:

<ParameterTable path="./*.yaml" keys={[".accuracy"]} />
