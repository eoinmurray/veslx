## Markdown basics
Use standard Markdown:

```md
**Bold**, *italic*, `code`

- Lists
- Tables

> Blockquotes
```

## MDX components
MDX lets you embed React components in Markdown.

```mdx
<MyComponent prop="value" />
```

## Imports
Use local components or npm packages:

```mdx
import { Counter } from './Counter.tsx'
import confetti from 'canvas-confetti'
```

## Tailwind classes
Tailwind works in both `.md` and `.mdx` files. Use `className` on JSX or `class` on HTML:

```mdx
<div className="rounded border px-4 py-2">
  Styled with Tailwind
</div>
```

```md
<div class="rounded border px-4 py-2">
  Styled with Tailwind
</div>
```

## Links
Internal links use relative paths:

```md
[Quickstart](./01-quickstart.mdx)
```

External links open in a new tab by default.

## TSX pages
You can place `.tsx` files in the content directory for fully custom pages. These always render unstyled (no Veslx layout wrappers).

```tsx
export const frontmatter = {
  title: "Custom Page",
  description: "Fully custom TSX page",
}

export default function CustomPage() {
  return (
    <div className="min-h-screen bg-black text-white p-12">
      <h1 className="text-4xl font-bold">Raw TSX</h1>
      <p className="mt-4 text-zinc-300">No layout, just your component.</p>
    </div>
  )
}
```
