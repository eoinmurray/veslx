# Example

<div className="mb-4 p-8 border rounded">
  <VeslxPostList globs={["example.1*", "example.2*"]} />
</div>

# Usage

```mdx
<VeslxPostList />
```

Props:

| Prop | Type | Default | Notes |
| --- | --- | --- | --- |
| globs | string[] | null | Filter by file glob |


Group content by globs (useful on a custom homepage):

```mdx
# Getting Started
<VeslxPostList globs={["00-*", "01-*"]} />

# Tutorials
<VeslxPostList globs={["tutorials/*"]} />
```
