## serve
Start the dev server:

```bash
veslx serve [directory]
veslx serve -p 8080
```

## build
Create a static build:

```bash
veslx build [directory]
veslx build -o ./public
```

## init
Generate a config file:

```bash
veslx init
```

## start / stop
Run as a background daemon:

```bash
veslx start
veslx stop
```

## Install
```bash
bun install -g veslx
npm install -g veslx
pnpm add -g veslx
```
