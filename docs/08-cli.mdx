# CLI

## `veslx serve`

Start dev server.

```bash
veslx serve [directory]
veslx serve -p 8080
```

## `veslx build`

Production build.

```bash
veslx build [directory]
veslx build -o ./public
```

Output is static, deploy anywhere.

## `veslx init`

Generate config file.

```bash
veslx init
```

## `veslx start`

Run as background daemon (pm2).

```bash
veslx start
```

## `veslx stop`

Stop daemon.

```bash
veslx stop
```

## Install

```bash
bun install -g veslx
npm install -g veslx
pnpm add -g veslx
```
